## 🎨 设计系统

### 颜色系统
```css
/* 角色颜色 */
--primary: 用户/选中状态
--warning: 主龙虾/待处理
--info: 工龙虾/进行中
--success: 质检员/已完成/健康
--error: 异常/删除操作

/* 渐变模板 */
from-{color}/15 to-{color}/5  /* 容器背景 */
from-{color} to-{color}/90    /* 按钮背景 */
from-{color}/10 to-{color}/5  /* 卡片激活背景 */
```

### 圆角系统
```css
rounded-lg: 8px    /* 小元素 */
rounded-xl: 12px   /* 按钮、输入框 */
rounded-2xl: 16px  /* 卡片、对话框 */
rounded-3xl: 24px  /* 大型装饰 */
```

### 阴影系统
```css
shadow-sm: 轻微阴影
shadow-md: 标准阴影
shadow-lg: 明显阴影（hover）
shadow-2xl: 对话框阴影
```

### 动画效果
```css
hover:scale-105        /* 按钮缩放 */
animate-pulse          /* 状态点脉冲 */
animate-in fade-in     /* 对话框淡入 */
zoom-in-95             /* 对话框缩放入场 */
opacity-0 group-hover:opacity-100  /* 渐显 */
```

## 🔍 技术细节

### 关键 CSS 技巧

1. **渐变叠加**
```vue
<div class="bg-gradient-to-br from-primary/15 to-primary/5">
  <!-- 内容 -->
</div>
```

2. **渐显按钮**
```vue
<button class="opacity-0 group-hover:opacity-100 transition-all">
  <span class="i-carbon-trash-can" />
</button>
```

3. **头像堆叠**
```vue
<div class="flex -space-x-1.5">
  <div v-for="member in members" class="w-6 h-6 rounded-full border-2 border-card">
    <!-- 头像 -->
  </div>
</div>
```

4. **入场动画**
```vue
<div class="animate-in fade-in zoom-in-95 duration-200">
  <!-- 对话框内容 -->
</div>
```

5. **脉冲动画**
```vue
<span class="w-2.5 h-2.5 rounded-full bg-success animate-pulse"></span>
```

### 响应式策略

- 使用 `flex` 和 `grid` 布局
- `min-w-0` 防止 flex 子元素溢出
- `max-w-[Npx]` 限制对话框和内容区域宽度
- `lg:grid-cols-N` 大屏幕多列布局
- `overflow-hidden` + `truncate` 处理长文本

### 性能优化

- CSS 文件大小控制在 120KB
- 使用 Tailwind 的 JIT 模式按需生成
- 图标使用 unplugin-icons 按需加载
- 避免不必要的 DOM 操作

## 🎓 学习要点

本次优化的核心设计理念：

1. **一致性优先**：所有页面使用相同的设计元素
2. **渐进增强**：基础功能先行，视觉效果锦上添花
3. **性能考虑**：使用 CSS 而不是 JS 动画
4. **可维护性**：使用 Tailwind 工具类，避免自定义 CSS
5. **可访问性**：保留 title、label、placeholder 等辅助信息
