# OpenClaws UI/UX 优化总结

## 优化概览

本次优化对整个 OpenClaws 应用进行了全面的视觉升级，确保所有页面保持统一的现代化设计语言。

## 设计原则

- **一致性**：所有页面采用统一的设计语言（Tailwind CSS 4 + Carbon Icons）
- **层次感**：使用渐变背景、阴影、圆角半径变化来营造视觉深度
- **交互反馈**：所有可交互元素都有 hover、active、disabled 等状态的视觉反馈
- **微动画**：使用 scale、shadow、opacity 等过渡效果提升操作体验
- **信息密度**：合理分配空间，确保信息清晰可读且不拥挤

## 关键设计元素

### 1. 圆角系统
- **小圆角 (rounded-lg/xl)**: 按钮、输入框、标签
- **大圆角 (rounded-2xl/3xl)**: 卡片、对话框、主要容器
- **圆形 (rounded-full)**: 徽章、状态点、头像

### 2. 渐变应用
- **背景渐变**: `bg-gradient-to-br from-{color}/15 to-{color}/5`
- **按钮渐变**: `bg-gradient-to-r from-primary to-primary/90`
- **阴影层次**: `shadow-sm` → `shadow-md` → `shadow-lg` → `shadow-2xl`

### 3. 颜色语义
- **Primary (主色)**: 用户操作、选中状态
- **Success (成功)**: 健康状态、完成状态、质检员角色
- **Warning (警告)**: 待处理、主龙虾角色
- **Info (信息)**: 进行中、工龙虾角色
- **Error (错误)**: 异常、删除操作

### 4. 图标使用
- **Carbon Icons**: 统一使用 Carbon Design System 的图标集
- **尺寸规范**: 
  - 页面标题图标: `text-2xl`
  - 卡片头像图标: `text-xl`
  - 按钮图标: `text-base`
  - 内联图标: `text-sm`

## 技术实现

### CSS 类命名规范

```css
/* 容器 */
.rounded-xl      /* 12px 圆角 - 小容器 */
.rounded-2xl     /* 16px 圆角 - 主要容器 */
.rounded-3xl     /* 24px 圆角 - 大型装饰 */

/* 间距 */
.gap-3          /* 12px - 紧凑间距 */
.gap-4          /* 16px - 标准间距 */
.gap-6          /* 24px - 宽松间距 */

/* 内边距 */
.p-4            /* 16px - 紧凑内边距 */
.p-5            /* 20px - 标准内边距 */
.p-7            /* 28px - 宽松内边距 */

/* 阴影 */
.shadow-sm      /* 轻微阴影 */
.shadow-md      /* 标准阴影 */
.shadow-lg      /* 明显阴影 */
.shadow-2xl     /* 对话框阴影 */
```

### 渐变背景模板

```vue
<!-- 角色头像 -->
<div class="bg-gradient-to-br from-{color}/15 to-{color}/5">
  <span class="i-carbon-icon"></span>
</div>

<!-- 按钮 -->
<button class="bg-gradient-to-r from-primary to-primary/90">
  操作
</button>

<!-- 卡片激活状态 -->
<div class="bg-gradient-to-br from-primary/10 to-primary/5">
  内容
</div>
```

### 动画效果

```vue
<!-- Hover 缩放 -->
<button class="hover:scale-105 transition-all">

<!-- Hover 阴影 -->
<div class="hover:shadow-lg transition-all">

<!-- 渐显按钮 -->
<button class="opacity-0 group-hover:opacity-100 transition-all">

<!-- 脉冲动画 -->
<span class="animate-pulse">

<!-- 入场动画 -->
<div class="animate-in fade-in zoom-in-95 duration-200">
```

## 响应式设计

所有页面都采用响应式设计，关键断点：
- `lg:grid-cols-4`: 大屏幕 4 列布局
- `max-w-md/lg/xl/2xl`: 对话框最大宽度限制
- `min-w-0`: 防止 flex 容器内容溢出
