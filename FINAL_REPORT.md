### 一、端口配置 (electron.vite.config.ts)

**问题**: 每次启动时 Vite 端口可能变化，导致开发不便

**解决方案**:
```typescript
renderer: {
  server: {
    port: 5173,
    strictPort: false,
    host: '127.0.0.1'
  }
}
```

**效果**: 
- ✅ 端口固定为 5173
- ✅ 避免端口冲突导致的困扰
- ✅ 开发体验更加一致
